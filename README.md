# Smart Agriculture IoT Dashboard

Project IoT monitoring tanah berbasis ESP32 yang menampilkan data pH tanah, kelembapan, suhu, serta kontrol otomatis pompa dan pupuk melalui dashboard web realtime.

---

## Fitur
- Monitoring pH tanah realtime
- Monitoring kelembapan tanah
- Monitoring suhu lingkungan
- Status pompa otomatis
- Status pupuk otomatis
- Grafik realtime
- Gauge meter pH premium
- Dashboard UI modern

---

## Arsitektur Sistem
ESP32 → ThingSpeak Cloud → Dashboard Web

---

## Teknologi
- ESP32
- ThingSpeak IoT Cloud
- HTML CSS JavaScript
- Chart.js
- Wokwi Simulator

---

## Sensor yang digunakan
- Sensor pH tanah
- Soil moisture sensor
- DHT22 (suhu)

---

## Cara menjalankan
1. Upload kode ESP32 ke perangkat / Wokwi
2. Pastikan data masuk ke ThingSpeak
3. Buka dashboard web
4. Dashboard akan update realtime setiap 15 detik
